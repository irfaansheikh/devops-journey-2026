# Week 1 – Linux Foundation

This week focuses on Linux basics, including file system navigation, commands, and initial Git usage.  
Folders:
- `linux-labs/` → hands-on exercises for Linux commands
- `docker-labs/` → introductory Docker exercises (setup, basic containers)
- `scripts/` → reusable Bash scripts
- `notes/` → daily reflections, issues, lessons, fixes

Daily notes are named by date (Day X - dd-mm-yyyy.md) to track progress.
